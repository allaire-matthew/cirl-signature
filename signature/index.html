Install in Gmail

  1. Click Copy Signature above
  2. Open Gmail as matthew@commonwealthirl.org
  3. Settings (⚙️) → See all settings → General
  4. Scroll to Signature → click + Create new → name it card
  5. Click in the signature text box and press ⌘V to paste
  6. Set both Signature defaults dropdowns to card
  7. Scroll down and click Save Changes

Gmail re-hosts the logo and icon on its own servers when you save, so the signature works forever even if this site goes down.
